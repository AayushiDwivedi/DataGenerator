DataGenerator is an Apache V2 licensed testing tool that allows the user to model the behavior of a piece of software by employing data specification and dependency modeling. Using templates, it enables the user to produce highly customizable and dynamically calculated test expectations, test data or test scripts to help test a piece of software.

Increased Coverage

Using combinatorial data generation and pair-wise data generation techniques it allows the user to cover boundary data test scenarios. By incoporating graph coverage techniques such as node, path and edge coverage it allows the user to capture application logic paths.

Decrease Manual Effort

By enabling the testers to template the output and model requirements, this tool reduces effort needed to hand craft test data to ensure coverage. This also makes it much easier for the testers to react to constant requirement changes that occur in a typical agile environment reducing overall time spent on test data creation and maintanance.

Flexible Output

Data that is generated can be applied multiple templates allowing the tester to have many types of output generated including test data, automation code, input data, verification data and verification scripts.
