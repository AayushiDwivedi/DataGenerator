DataGenerator
is an automated tool designed to ease the pain of test creation and maintenance.

Download Latest Executable »

Get involved on GitHub


Increased Coverage

DataGenerator generates test data using combinatorial coverage techniques like 'pairwise combinations' and graphical coverage techniques like 'all paths'.

Flexible Output

Leveraging state-of-the-art templating, DataGenerator can produce test input, expected results, and automation scripts in any textual format.

Decreased Manual Effort

DataGenerator automates test maintenance, allowing you to work smarter and respond to change faster in an Agile environment.

 
Quick Start
  1. Download Latest DataGenerator Download Latest Executable »

  2. Make sure you have the latest JRE/JDK installed on your machine. If not, go to http://www.oracle.com/technetwork/java/javase/downloads/index.html to download it.

    3. Logic modelling example:

    1. Get this download » of a sample data generator project.
    2. Expand it. You will see the following folder structure.




      3. 'input' folder contains the branch modeling specifications.
      'input/branchspec' folder contains the visio dependency models.
      'input/dataspec' folder contains the parameter definitions.
      'input/teamplate' folder contains the DataGenerator output templates.
      'dd.properties' file within the input folder contains global parameter for DataGenerator.

  3. Either:

    4. - Execute GC.bat file from the Command shell.
    CMD /c C:\DT\GC.bat

    OR

    - Execute the following java command from the zip extraction folder.
    java -jar DataDynamo-1.0-SNAPSHOT.jar -dataspec input\dataspec -branchgraph input\branchspec -templates input\template -out out -allPaths -config input\dd.properties

    - 'out' folder will have your generated output.
