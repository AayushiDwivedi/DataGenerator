Quick Start
===========

This section provides the steps to quickly get started with Data Generator. Before you start, make sure you have the latest version of DataDynamo and the JDK. Use the links below in case you don't have them setup.

1. Download the latest version of the DataGenerator, here's a link to the zip file from the master branch: https://github.com/FINRAOS/DataGenerator/archive/master.zip
2. Install the latest JDK http://www.oracle.com/technetwork/java/javase/downloads/index.html

The function of DataGenerator is to generate test scripts, that should help the user acheive a higher level of automation.
